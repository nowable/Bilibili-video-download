# Bilibili Video Downloader / B 站视频下载器

一个简单易用的 B 站视频下载工具，支持通过网页界面或命令行下载 B 站视频。

## ✨ 功能特性

- 🎬 支持下载 B 站普通视频和分享链接
- 📹 自动提取最高画质视频
- 🌐 提供友好的 Web 界面
- 💻 支持命令行模式
- ⚡ 实时显示下载进度
- 🔒 本地运行，无需上传任何数据

## 🚀 快速开始

### 方法一：Web 界面（推荐）

1. 运行程序
2. 在浏览器打开 `http://localhost:5000`
3. 粘贴 B 站视频链接
4. 点击"开始下载"

### 方法二：命令行

```bash
python bilibili_downloader.py
```

修改脚本中的 `video_url` 为你要下载的视频链接。

## 📦 依赖

- Python 3.x
- Flask
- bilibili-api
- requests
- flask-cors

安装依赖：
```bash
pip install -r requirements.txt
```
